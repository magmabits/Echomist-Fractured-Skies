This mods adds a new material, Sylvite, which focused on a healing-oriented playstyle!

### Features
- Sylvite Halberd, a weapon which inflicts "Afflicted". This effect reduces the target's healing efficiency.
- Sylvite Token, a utility item that heals the user.
- Budding Sylvite, which grows Sylvite Shards
- Sylvite Blocks (Including Stairs, Slabs and Walls)
- Sylvite Bricks (Including Stairs, Slabs and Walls)
- Sylvite Tiles (Including Stairs, Slabs and Walls)
- Chiseled Sylvite
- Sylvite Pillar

Sylvite Shards can also be used as armor trim materials!
